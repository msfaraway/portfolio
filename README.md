# Farhana Rayhan Andar — Portfolio

Personal portfolio site for Farhana Rayhan Andar, WordPress Developer, Graphic Designer & Automation Specialist.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `farhana-portfolio`)
2. Upload `index.html` to the root of the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)` → Save
5. Your site will be live at `https://yourusername.github.io/farhana-portfolio`

## 📝 Customization

Before uploading, update these in `index.html`:
- Replace LinkedIn and Personal Website URLs in the **Contact** section
- Add your real LinkedIn URL and personal website URL

## 🛠 Tech
Pure HTML/CSS — no build tools, no dependencies. Works out of the box.
